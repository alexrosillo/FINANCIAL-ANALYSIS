# Advanced Financial Analysis & Probabilistic Modeling of Big Tech Stocks

This project provides a quantitative analysis of three mega-cap technology equities (Apple, Microsoft, and Google) using advanced mathematics and statistics. By analyzing historical price trends and executing formal econometric tests, the study evaluates asset distributions and risk profiles to model future scenarios.

## 📊 Key Features

* **Historical Analysis:** 3-year performance tracking and data normalization.
* **Statistical Testing:** Evaluation of return distributions, skewness, and kurtosis to challenge market normality.
* **Stochastic Modeling:** 2,000-path Geometric Brownian Motion Monte Carlo simulations for predictive risk control.
* **Strategic Allocation:** Data-driven insights for institutional portfolio management and tail-risk mitigation.

## 📁 Repository Structure

* `README.md` - Core project documentation and overview.
* `data/` - Historical market datasets.
* `notebooks/` - Python/R scripts, statistical calculations, and Monte Carlo simulation models.
* `report/` - Final financial documentation.
